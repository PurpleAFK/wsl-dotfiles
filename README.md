# dotfiles

## Run Locally

Install all the necessary tools

```bash
  fzf
  fzf-git
  zoxide
  eza
  yazi
  tmux
  ripgrep
  python3
  npm
  fd

```

Install Stow

```bash
  sudo pacman -S stow
```

Clone the project

```bash
  git clone https://github.com/PurpleAFK/dotfiles.git
```

Go to the project directory

```bash
  cd dotfiles
```

Use Stow to create the symlinks

```bash
  stow .
```
