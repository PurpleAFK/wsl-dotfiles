# dotfiles

## Run Locally

Install dependencies (if needed)

```bash
  sudo pacman -S npm
  sudo pacman -S stow
```

Clone the project

```bash
  git clone https://github.com/PurpleAFK/dotfiles.git
```

Go to the project directory

```bash
  cd dotfiles
```

Use Stow to create the symlinks
```bash
  stow .
```
